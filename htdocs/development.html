Development

REPORTING BUGS

Fully descriptive bug reports and feature requests may be posted via the "new ticket" link on the wiki, after logging in as "guest"/"guest". Please review the current list of reports before posting.

DEVELOPMENT DISCUSSION

Most development discussion occurs on the main mailing list or IRC channel - further info is available on the community page.

Tracking Activity

SVN Commits can be viewed via the timeline (rss feed) which also mails to the Commit List. Ticket changes are posted to the Ticket List.

SQLAlchemy has a buildbot master at http://buildbot.sqlalchemy.org/.

A summary of the most recent features and bugfixes in trunk can always be viewed at CHANGES.

Current Trunk - 0.6

The current development version of SQLAlchemy is 0.6.

The latest trunk of SQLAlchemy is available at:

svn checkout http://svn.sqlalchemy.org/sqlalchemy/trunk sqlalchemy

Development Mirrors

A (sometimes lagged) Mercurial mirror of the trunk is available on bitbucket.org:

hg clone http://bitbucket.org/mirror/sqlalchemy/

Previous Maintenance Branch

Version 0.5:

svn checkout http://svn.sqlalchemy.org/sqlalchemy/branches/rel_0_5 sa05

Note that maintenance branches must maintain 100% backwards compatibility with previous releases in that series. No new feature requests are accepted for maintenance branches.
