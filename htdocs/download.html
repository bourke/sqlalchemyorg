Development

The current development version of SQLAlchemy is 0.6.

A document in describing major changes in the 0.6 release is in 06Migration. Please read this document for guidelines on porting 0.5 code to 0.6, as some APIs have changed. Please do not post bug reports without first being familiar with these changes !

the latest trunk of SQLAlchemy is available at:

svn checkout http://svn.sqlalchemy.org/sqlalchemy/trunk sqlalchemy

A Mercurial mirror of the trunk is also available on bitbucket.org:

hg clone http://bitbucket.org/mirror/sqlalchemy/

0.5 Maintenance Branch

A document in describing the 0.5 release is in 05Migration.

0.5 is now in maintenance mode and is available via a branch:

svn checkout http://svn.sqlalchemy.org/sqlalchemy/branches/rel_0_5 sa05

Note that maintenance branches must maintain 100% backwards compatibility with previous releases in that series. No new feature requests are accepted for maintenance branches.
